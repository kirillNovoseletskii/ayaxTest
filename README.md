# ayaxTest
Это выполненное тестовое заданин для компании Ayax.
C использованием Vue и библиотеки nuxt.js.
## Build Setup

```bash
# установить зависимости
$ yarn install

# Проверка работоспособности
$ yarn build
$ yarn start
# Зайти с любого устройства
http://0.0.0.0:3000/
#где 0.0.0.0 - свободный ip
```
